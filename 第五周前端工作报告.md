### 2.4 Intervention 干预建议页 🟡 重要

`pages/Intervention/index.tsx` + 子组件

**页面结构：**
- **类型选择区**：5个类型圆角按钮（🏔️畏难型 / 😰焦虑型 / 🎮贪玩型 / 🗺️无规划型 / ✨完美主义型），点击选中紫色高亮
- **AI 生成按钮**：渐变紫全宽按钮 → 调用 `generateIntervention(type)` → DeepSeek 生成个性化方案
- **建议列表**：`SuggestionList` 组件渲染 AI 生成的 `SuggestionCard` 列表
- **预设策略库**：无 AI 结果时展示 `getAllStrategies()` 返回的 5 类基础策略
- **每日提醒**：AI 生成过建议后出现，可设置每日提醒时间

**数据流：**
```
选择类型 → generateIntervention(type) → 后端 DeepSeek → 返回 Suggestion
         → getAllStrategies()           → 后端预设库   → 返回 StrategySummary[]
```

## 五、技术要点

### 5.1 三种按钮动效

| 动效 | 用途 | 实现 |
|------|------|------|
| `ripplePulse` | 打卡按钮涟漪 | 3层 `border` 从 80px→160px 扩散 + opacity 渐消 |
| `checkBounce` | 已打卡图标 | 0→1.2→1 scale 弹跳 |
| `glowPulse` | 徽章光晕 | 1→1.1 scale + opacity 呼吸 |

### 5.2 微行动随机池

8条建议用 Fisher-Yates 思想简化：每次随机选一条，若与当前相同则重选，保证新鲜感。

### 5.3 干预页双数据源

- **AI 模式**：`generateIntervention(type)` → DeepSeek 生成个性化方案
- **离线模式**：`getAllStrategies()` → 预设基础策略库，即时返回不调 AI

### 5.4 打卡月历日期计算

使用 dayjs 计算当月首日→月末 + 前后补齐完整周数，非当月日期 opacity: 0.25 区分。
