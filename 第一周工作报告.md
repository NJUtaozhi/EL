### 2.2 全局样式 — 治愈系色板

**`src/styles/variables.css`** — 定义了完整的设计令牌（Design Tokens）：

- **主色调**：莫兰迪色系（柔和蓝 `#A8D8EA` / 柔和紫 `#AA96DA` / 柔和粉 `#FCBAD3` / 暖黄 `#FFFFD2` / 薄荷绿 `#B5EAD7`）
- **功能色**：成功/警告/错误/信息
- **文字色**：主文字 `#4A4A4A` / 次要 `#8C8C8C` / 浅色 `#BFBFBF`
- **背景色**：暖白底色 `#FAF9F6` / 白色卡片 / 输入框背景
- **阴影**：4级（sm/card/hover/modal）
- **圆角**：3级（sm=8px / md=12px / lg=20px / full=9999px）
- **间距**：5级（4px ~ 32px）
- **字体**：系统字体栈，优先苹方/微软雅黑，5级字号
- **布局常量**：导航栏高度 48px / 标签栏高度 56px
- **动画时长**：3级（fast=0.15s / normal=0.25s / slow=0.35s）
- **图表色板**：6色渐进色板，与主色调一致

**`src/styles/global.css`** — 全局样式重置和工具类：

- CSS Reset（盒模型、边距、字体）
- 根容器：`max-width: 480px` 居中（移动优先）
- 滚动条美化（4px 宽度，圆角）
- `.page-container` 页面容器（自动留出底部 TabBar 空间）
- `.card` 卡片通用样式（白底 + 圆角 + 阴影 + 点击动效）
- `.healing-gradient` 治愈系渐变
- 6 个关键帧动画：`fadeIn` / `scaleIn` / `slideUp` / `pulse` / `checkBounce`

## 三、技术选型与设计决策

| 决策 | 理由 |
|------|------|
| CSS Modules（非 styled-components） | 文档要求每个页面配 `style.module.css`，保持一致性 |
| 自定义 TabBar（非 TDesign TabBar） | 需要毛玻璃效果 + 治愈系动画，TDesign 组件定制性不足 |
| 页面占位提前创建 | 保证路由可访问、NavBar 标题映射完整，后续直接填充 |
| `max-width: 480px` 居中布局 | 移动优先设计，与文档 UI 设计规范一致 |
| CSS 变量全部放在 `:root` | 支持运行时主题切换（后续 TDesign 主题定制） |
