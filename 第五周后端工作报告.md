## 二、已完成工作详情

### 2.1 干预方案服务 — [services/interventionService.ts](backend/src/services/interventionService.ts)

**核心方法**：

| 方法 | 说明 |
|------|------|
| `generateIntervention(userId, request)` | ⛓️ 核心干预：取用户画像 → 组装 Prompt → 调 DeepSeek → 解析 JSON → 返回建议 |
| `getPresetStrategy(type)` | 获取指定类型预设策略（不调 LLM，即时返回） |
| `getAllStrategies()` | 获取全部 5 种拖延类型的策略摘要 |

**干预 Prompt 工程**（包含完整的 CBT 策略库）：

```
System Prompt 结构：
├── 角色定位：拖延心理学顾问（CBT + 时间管理 + 行为激活）
├── 5种类型的核心干预策略（每类4条具体策略）
│   ├── 畏难型：任务拆分法 / 微行动启动 / 进度可视化 / 5分钟法则
│   ├── 焦虑型：认知重构 / 担忧时间法 / 呼吸放松法 / 最坏情况分析
│   ├── 贪玩型：番茄工作法 / 环境设计 / 替代奖励 / 诱惑捆绑
│   ├── 无规划型：ABC优先级法 / 每日三件事 / 时间块规划 / 晚间复盘
│   └── 完美主义型：MVP原则 / 设定够好标准 / 时间盒法 / 80分法则
├── 输出规范：JSON 格式 {type, strategy, title, steps[3], encouragement, tip}
└── 个性化要求：结合用户实际任务，非泛泛鸡汤
```

**User Prompt 构建**（含用户画像）：

```
【拖延类型】畏难型
【当前任务】写期末论文
【用户画像】
- 累计记录任务数：15
- 完成归因分析次数：5
- 常见情绪：焦虑、沮丧
- 近期拖延类型：畏难型、焦虑型
```

**LLM 输出示例**：
```json
{
  "type": "畏难型",
  "strategy": "任务拆分法",
  "title": "将大山变成台阶，一步一步走",
  "steps": [
    "第1步：将「写期末论文」拆分为选题→提纲→每章草稿→排版，每个子任务不超过30分钟",
    "第2步：从最简单的「列提纲」开始，设定25分钟番茄钟完成初稿",
    "第3步：每完成一个子任务在清单上打勾，积累3个勾后奖励自己休息10分钟"
  ],
  "encouragement": "论文不是一天写完的，每个学者都是从第一行字开始。你不需要今天就看到终点，只需要看清接下来25分钟要做什么。",
  "tip": "试试把手机放在另一个房间，用纸笔先列出3个要点"
}
```

## 五、核心数据流

```
用户在前端 Intervention 页面选择拖延类型
  │
  ▼
POST /api/intervention/generate  { procrastinationType: "畏难型", taskTitle?: "写论文" }
  │
  ▼
routes/intervention.ts → auth → Zod 校验类型枚举
  │
  ▼
controllers/interventionController.generateIntervention()
  │
  ▼
services/interventionService.generateIntervention(userId, request)
  │
  ├─ 1. 校验类型有效性 → 无效则 400
  │
  ├─ 2. 并行获取用户画像
  │     ├── Prisma: 任务总数
  │     ├── Prisma: 最近10次分析类型
  │     ├── Prisma: 最近10次任务情绪
  │     └── 提取：commonEmotions + recentTypes
  │
  ├─ 3. 构建 User Prompt（拖延类型 + 任务名 + 用户画像）
  │
  ├─ 4. DeepSeek API 调用
  │     ┌───────────────────────────────────────────┐
  │     │ System: CBT专家 + 5类型完整策略库          │
  │     │ User:   拖延类型 + 用户画像 + 具体任务     │
  │     │ temperature: 0.7 (多样化建议)              │
  │     │ response_format: json_object               │
  │     └───────────────────────────────────────────┘
  │     │
  │     ▼ 返回 JSON
  │     { type, strategy, title, steps[], encouragement, tip }
  │
  ├─ 5. parseInterventionJSON(raw) → 容错解析
  │     └─ 失败 → throw 502
  │
  └─ 6. 返回 Suggestion（实时生成，不持久化）
        └─ baseStrategy 附带预设策略模板
```

**预设策略快捷路径**（不调 LLM）：

```
GET /api/intervention/strategy/畏难型
  → interventionService.getPresetStrategy("畏难型")
    → 匹配 INTERVENTION_STRATEGIES["畏难型"]
      → 即时返回 { type, baseStrategy, isValid: true }
        (响应时间 < 5ms)
```
