## 五、技术要点

### 5.1 三种 ECharts 图表模式

| 图表 | 类型 | 用途 |
|------|------|------|
| 饼图 (Pie) | 环形空心 `radius: ['55%', '78%']` | 拖延类型分布 |
| 热力图 (Heatmap) | 矩阵色块 + visualMap | 星期×时段拖延频率 |
| 折线图 (Line) | 平滑 + 渐变面积 | 周任务/打卡趋势 |

### 5.2 Dashboard API 分层

- `getDashboardOverview()`：首页用，返回轻量聚合数据
- `getDashboard()`：画像页用，返回含周趋势的完整数据

### 5.3 打卡防重复

`useCheckin` Hook 的 `loading` 状态 + 按钮 `disabled` + zustand store 的 `todayCheckedIn` 三重保障。

### 5.4 热力图模拟数据

热力图数据需要用户的详细时段数据（目前后端 dashboard 尚未提供此维度），前端以 `generateMockHeatMap()` 生成模拟数据展示效果，待后续接入真实接口。
