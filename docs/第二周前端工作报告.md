#### `hooks/useVoiceInput.ts` 🟡 重要

浏览器语音识别封装：

- **状态机**：`idle → listening → idle/error/unsupported`
- 自动检测浏览器是否支持 `SpeechRecognition` API

### 2.4 Log 页面完整实现

![Log页面结构]

| 区域 | 说明 |
|------|------|
| 页面标题 | "📝 拖延日志" + 副标题 |
| 新建表单区 | `TaskForm` 组件（折叠/展开） |
| 列表工具栏 | "记录列表(N)" + 刷新按钮（loading时旋转） |
| 加载态 | 卡片内居中提示"加载中..." |
| 空状态 | 📋图标 + "还没有拖延记录" + 引导文字 |
| 错误态 | 错误信息 + "重试"按钮 |
| 任务列表 | `TaskCard` 列表，依次淡入动画 |

数据流：

```
用户输入 → TaskForm → createTask() → 后端 API
                  → addTask() → zustand store → 列表刷新
                  
页面加载 → getTasks() → setTasks() → 渲染列表

删除操作 → deleteTask() → removeTask() → 列表更新

点击卡片 → navigate(/analysis?taskId=id) → 跳转分析页
```

## 四、技术要点

### 4.1 表单折叠模式

TaskForm 使用折叠/展开的交互模式：默认显示一个按钮节省空间，点击后才展开完整表单。提交成功后自动收起，保持页面整洁。

### 4.2 语音识别兼容性

`VoiceInput` 组件启动时自动检测浏览器 `SpeechRecognition` API 是否可用，不可用时显示禁用态而不崩溃。

### 4.3 状态管理集成

所有数据流经 zustand store 统一管理，API 调用成功后同步更新本地状态，保证 UI 即时响应。

### 4.4 动画细节

- 页面进入：`fadeIn` 淡入
- 表单展开：`scaleIn` 弹入
- 列表卡片：依次延迟 `0.05s * index` 淡入
- 录音状态：三层波纹扩散动画
- 刷新按钮：`spin` 旋转动画
- 选中指示点：`scaleIn` 弹入
